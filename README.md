# Sales Insights Dashboard (Power BI)

## Project Overview
This project focuses on analyzing historical sales data to generate actionable business insights using **Power BI**.  
The dashboard enables stakeholders to track revenue trends, sales quantity, market performance, and customer/product contribution across multiple years.

**Project Timeline:** November 2025  
**Domain:** Sales Analytics / Business Intelligence

---

## Problem Statement
Businesses often struggle to understand sales performance across markets, customers, and products due to scattered and unstructured data.  
The goal of this project is to transform raw sales data into an interactive dashboard that supports **data-driven decision-making**.

---

## Data Source
- Relational sales database (SQL dump provided)
- Data includes:
  - Orders
  - Customers
  - Products
  - Markets
  - Dates

---

## Data Processing & Modeling
- Imported SQL data into Power BI
- Cleaned and transformed data using **Power Query**
- Created relationships between fact and dimension tables
- Built calculated measures using **DAX** for KPIs

---

## Key Metrics (KPIs)
- **Total Revenue:** 984.88M  
- **Total Sales Quantity:** 2M  

---

## Dashboard Features
- Revenue and sales quantity by **market**
- Time-series analysis by **year, quarter, month**
- Top 5 customers by revenue
- Top 5 products by revenue
- Interactive slicers for:
  - Year
  - Month
  - Date hierarchy
